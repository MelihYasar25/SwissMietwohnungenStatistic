🇨🇭 Swiss Mietwohnungen Statistik

This project is a web application for exploring and analyzing rental apartments (Mietwohnungen) in Switzerland.
It provides powerful search tools, statistical insights, and an interactive quiz that recommends apartments based on user preferences.

The goal of this project is to make it easier to understand the Swiss rental market, compare apartments, and discover suitable housing options.

✨ Features
🔎 Apartment Search

Search rental apartments across Switzerland

Filter by:

Canton

City

Price range

Apartment size (m²)

Number of rooms

Property type

Sort results by price, size, or newest listings

View apartments in cards or list view

📊 Statistics Dashboard

Average rent in Switzerland

Average rent by canton and city

Price per square meter

Distribution by apartment size and rooms

Regional comparisons

🎮 Apartment Finder Quiz

Interactive quiz that recommends apartments based on user preferences:

Preferred canton or city

Budget

Apartment size

Number of rooms

Lifestyle preferences

The system analyzes the answers and suggests the best matching apartments.

⭐ Favorites System

Save apartments to favorites

Stored locally in the browser

Quick access to saved listings

⚖️ Apartment Comparison

Compare multiple apartments

Price

Size

Rooms

Location

Price per m²

It also showcases:

Data filtering and search

Frontend statistics

Interactive recommendation systems
