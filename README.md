# 🇨🇭 Swiss Mietwohnungen Statistik

A modern web application for exploring, filtering, and analyzing **rental apartments in Switzerland**.  
The project includes a **statistics dashboard**, **advanced search filters**, **favorites**, **comparison tools**, and an interactive **Wohnungsfinder quiz** that recommends apartments based on user preferences.

---

## ✨ Features

- 🔎 Search and filter Swiss rental apartments
- 📊 View housing market statistics and trends
- 🎮 Answer a quiz and get apartment recommendations
- ⭐ Save apartments to your favorites
- ⚖️ Compare multiple apartments side by side
- 📱 Fully responsive design for desktop, tablet, and mobile
