# 🇨🇭 Swiss Mietwohnungen Statistik

A modern web application for exploring, filtering, and analyzing **rental apartments in Switzerland**.
The project provides an interactive way to discover housing options while gaining insights into the Swiss rental market.

---

## ✨ Features

* 🔎 **Advanced Search** – Find rental apartments using powerful filters such as location, price, and size.
* 📊 **Statistics Dashboard** – Explore housing market data and trends across Switzerland.
* 🎮 **Wohnungsfinder Quiz** – Answer a short quiz to receive apartment recommendations tailored to your preferences.
* ⭐ **Favorites System** – Save apartments and revisit them later.
* ⚖️ **Apartment Comparison** – Compare multiple apartments side-by-side to make better decisions.
* 📱 **Responsive Design** – Optimized for desktop, tablet, and mobile devices.
